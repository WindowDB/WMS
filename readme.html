采购及到货上架

零售商向供应商发起采购单供应商根据采购单发货，到货后库房作业人员质检收货，并按收货批次上架的全部流程。参考《采购入库作业指导》  关于托盘码以及物流箱标签的打印，请参考文档《库房规划作业指导》
  以下是供应商采购流程涉及的全部账户的登录码信息，参考《零时登陆码使用说明文档》：
 


 复制某一用户的临时登陆码后，访问在线演示系统登录页（IE浏览器不能打印单据，建议使用谷歌），点击左下角“快速登录【Alt+S】”或者按“【Alt+S】”呼出临时登陆码输入框，将复制的字符串粘贴到输入框中敲一下回车键即可实现登录。库房作业人员（负责上架的叉车司机）在使用RF终端作业的时候，使用RF枪自带的浏览器（支持IE6）访问 RF终端系统将复制的临时登录码粘贴到登录框中，按【ENT】回车键即可登录。
